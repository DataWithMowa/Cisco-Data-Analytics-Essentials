When performing any kind of data experiment or analysis, it is critical to define the key characteristics that need to be measured or observed. These characteristics to be studied are called variables. A variable in this context is anything that varies from one instance to another, that can be measured, and whose value can be manipulated or controlled in theoretical scenarios.
The recordings of the values, patterns, and occurrences for a set of variables are observations. 

The value or set of values for a specific observation is called a data point. Each observation can be thought of and represented as a record in a database or a row in an Excel spreadsheet. The collection of observations makes up the dataset for your analysis.
Observations usually have a purpose, and the variables included will depend on their relevance to that purpose.

# Practical Examples: Variables, Observations, and Data Points

To help solidify these concepts, it’s useful to look at them through different "lenses"—from a grocery store to a clinical trial. In every case, remember: **the Variable is the question, the Observation is the subject, and the Data Point is the specific answer.**

---

## 1. The Used Car Lot (Business Example)
If you are shopping for a car, you are subconsciously analyzing a dataset.

* **Variables:** These are the attributes you care about for every car (Year, Make, Model, Mileage, Price, Color).
* **Observation:** This is a **single car** on the lot. Each car is one complete row of information.
* **Data Point:** This is a specific value. For example, "2019" is the data point for the *Year* variable for "Car #4."

| Car (Observation) | Year (Variable) | Price (Variable) |
| :--- | :--- | :--- |
| Car #1 | 2021 | $24,000 |
| Car #2 | 2018 | $15,500 |

## 2. The Medical Check-up (Healthcare Example)
Imagine a doctor’s office keeping records of patients visiting for a yearly physical.

* **Variables:** Height, Weight, Blood Pressure, Heart Rate. These are the "labels" at the top of the clipboard.
* **Observation:** This is the **individual patient** (e.g., "John Doe"). The entire check-up result for John is one observation.
* **Data Point:** John’s specific weight of "180 lbs." This single number is the data point.

---

## 3. The "Library Book" Analogy
Think of a library’s digital catalog system.

* **Variables:** The categories of information the library tracks for every book: *Title, Author, ISBN, Genre, Status (Checked in/out).*
* **Observation:** A **specific book** (e.g., "The Great Gatsby"). All the info about this one book constitutes one record.
* **Data Point:** "F. Scott Fitzgerald." This is the specific value found in the *Author* column for that specific book.

---

## 4. The Weather Station (Scientific Example)
Meteorologists track the environment every hour.

* **Variables:** Temperature, Humidity, Wind Speed, Precipitation.
* **Observation:** A **specific moment in time** (e.g., "Monday at 12:00 PM").
* **Data Point:** "10 mph." This is the value for *Wind Speed* recorded at that exact time.
