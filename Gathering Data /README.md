
# Data Collection Fundamentals

This folder contains comprehensive information regarding the initial stages of data analysis, focusing on how we identify and categorize the information we collect.

## Key Topics Covered
* **Gathering Data:** The process of defining the purpose of your study and identifying relevant sources of information.
* **Observations, Variables, and Values:** The three core building blocks of any dataset.
    * **Variables:** The characteristics being measured (The Columns).
    * **Observations:** The individual subjects or events (The Rows).
    * **Values:** The specific data points recorded (The Cells).
* **Types of Variables:** Understanding the difference between Qualitative (Categorical) and Quantitative (Numerical) data.

## Learning Objectives
By reviewing the materials in this folder, you'll see that I have been able to:
1. Distinguish between the structure of a variable and an observation.
2. Correctly identify the "Data Point" within a specific dataset.
3. Classify data into appropriate categories (Nominal, Ordinal, Discrete, or Continuous) to determine the correct analysis method.
