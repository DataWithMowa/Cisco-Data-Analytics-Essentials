# 📖 What is this project about?
Sometimes, data is too big to fit into just one Excel file. Other times, different people keep their own files (like a Sales team and an Accounting team), and you need to bring their numbers together.

This project teaches me how to **link** different Excel files so they "talk" to each other. When you change a number in one file, the main report updates automatically.

---

### 🎯 What I learnt
*   **Connecting Files:** How to pull data from one workbook into another without copying and pasting.
*   **Keeping Data Fresh:** How to refresh your main file so it always shows the latest information.

---

### 🛠 Why use this method?
*   **Bring everything together:** Combine data from different years or departments into one simple summary.
*   **Show only what matters:** Create a small report that pulls only the important parts from a massive data file.
*   **Stay organized:** Instead of having one "giant" slow file, you can have several smaller, faster files that work together.

---

### 🚀 The Plan
1.  **Part 1:** Create links between different workbooks.
2.  **Part 2:** Learn how to update those links when the original data changes.

---

> Think of this project like a **live feed**. Instead of taking a "picture" of the data (copy-pasting), you are creating a "window" that always shows what’s happening in the other file.
