# Excel Projects: Cisco Data Analytics Essentials

Welcome to the Excel Projects repository. This folder serves as a comprehensive portfolio of my learning journey, practical lab applications, assignments, and statistical evaluations completed during the Cisco Data Analytics Essentials course. 

The projects contained herein demonstrate my proficiency in data cleaning, structured manipulation, advanced lookups, data visualization, and statistical modeling using Microsoft Excel.

---

## 📂 Folder Structure & Contents

### 1. 📖 Core Learning & Methodology
Notes and core frameworks detailing data analysis best practices, including data governance, structural integrity, and choosing the right analytics tools for specific business intelligence objectives.

### 2. 🧪 Lab Projects
Hands-on technical exercises completed using guided datasets to master specific Excel functions and analytics tools:
* **Lab - Create Visualizations in Excel:** Built dynamic data visualizations (Line with Markers, Stacked Column, and 2D-Pie charts) to highlight revenue trends, annual profits, and demographic breakdowns.
* **Lab - Interpret Visualizations with Respect to Outliers:** Utilized Sort techniques, Scatter Plots, and advanced statistical functions (`LARGE` and `SMALL`) to locate, analyze, and safely handle anomalous data points.
* **Lab - Using VLOOKUP in Data Analysis:** Implemented structured vertical lookups across thousands of rows of business data, incorporating error handling (`IFNA`) to streamline data extraction.

### 3. 📝 Practice Assignments
Independent practice scenarios designed to solve practical business problems, refine data cleaning workflows, and establish robust data auditing mechanisms (such as using nested `IF` and `ISNA` functions to isolate duplicate information).

### 4. 📊 Statistical Exam
A comprehensive assessment evaluating my knowledge of inferential statistics and advanced modeling tools. This project specifically covers the application and interpretation of **Regression Analysis** to establish variable relationships, determine model reliability ($R^2$), and forecast outcome metrics.

---

## 🛠️ Skills and Tools Demonstrated

* **Data Cleaning & Auditing:** Identifying duplicates, managing leading zeros using string formatting (`'`), and mitigating formula errors (`#N/A`).
* **Advanced Lookup Formulas:** Mastery of `VLOOKUP` constraints, `XLOOKUP` bi-directional flexibility, and alternative `INDEX` / `MATCH` arrays.
* **Data Visualization:** Customizing chart elements, adjusting number formats, altering legend alignments, and plotting scatter charts for anomaly identification.
* **Inferential Statistics:** Understanding how regression analysis can be leveraged to draw broader population inferences from localized sample datasets.

---
*All laboratory activities were performed and validated using Microsoft 365 Excel.*
