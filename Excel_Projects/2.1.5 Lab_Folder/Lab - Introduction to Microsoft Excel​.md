## Background / Scenario
**Microsoft Excel** is a spreadsheet application developed by Microsoft to run on Windows, Mac OS, Android, and iOS. Excel includes calculation and computation capabilities, as well as tools for creating graphs, charts, and pivot tables.

Microsoft provides a free online version of their Office suite at **Office.com**, which includes Excel. The free version does have some limitations. It does not have the full functionality and features of the paid versions, and it can only be used while connected to the Internet.

> **Note:** There are alternative free spreadsheet applications such as **Google Sheets** and **Apache Calc**. While the steps in this lab were created for the free version of Excel from Office.com, other spreadsheet applications have the same functionalities and similar interfaces and can be used to accomplish this work.

## Part 3: Working with Cells, Rows, and Columns

An Excel worksheet is made up of a set of **rows** and **columns**. Rows are identified by **numbers** and columns are identified by **letters**. An Excel worksheet has 16,384 columns and 1,048,576 rows.

The rectangular box at the intersection of a row and column is called a **cell**. Each cell is identified by its column and row coordinates, such as `A1` or `B2`. The coordinates of the active cell are indicated in the **Cell Name Box** above column A. A cell reference is always the column letter followed by the row number.  Values, which include numbers and letters, can be typed into cells. Cells can also contain mathematical formulas and functions for performing calculations.

### Step 1: Changing the Active Cell

1. Click inside cell **A1** at the top left corner of the worksheet. Notice the cell now has a border around it and the cell coordinates are displayed above column A in the **Cell Name Box**.
2. This indicates that cell A1 is the **active cell**. Only one cell is active at a time and data can only be entered into the currently active cell.
3. Now, click into cell **B2**. Note that this cell is now the active cell with a border and its reference coordinates are in the **Cell Name Box** above column A.
